# Simple Calculator

A simple, functional calculator built with HTML, CSS, and JavaScript.
MADE BY GITHUB COPILOT

## Features

- ✅ Basic arithmetic operations (addition, subtraction, multiplication, division)
- ✅ Clear function to reset the calculator
- ✅ Sign toggle (+/-) for positive/negative numbers
- ✅ Decimal point support
- ✅ Modern, clean UI with gradient background
- ✅ Interactive hover effects
- ✅ Division by zero protection

## How to Use

1. Open `calculator.html` in your web browser
2. Click the number buttons to input values
3. Select an operation (+, -, ×, ÷)
4. Click the equals button (=) to calculate the result
5. Use the C button to clear and start over

## Technology Stack

- **HTML**: Structure and layout
- **CSS**: Styling and responsive design
- **JavaScript**: Calculator logic and interactivity
